\begin{block}{Contributions}

This work introduces:

\begin{enumerate}

\item
A tracking-integrated InISAR framework for reconstructing
high-quality radar-based human point clouds.

\item
A spatial-temporal transformer architecture for learning
identity-discriminative radar representations.

\item
A pre-trained meta-learning strategy enabling few-shot
adaptation to unseen subjects.

\end{enumerate}


\textbf{Future directions}

\begin{itemize}
    \item Larger-scale radar datasets
    \item Online adaptation for real-world deployment
    \item Multi-modal sensor fusion
\end{itemize}

\end{block}
